# zippity
Creates Zip Files, quickly.

---

## Getting Started

Complete [onboarding](https://github.com/groundbreaker/codex/wiki/Onboarding), then:

    git clone groundbreaker/REPOS_NAME ${GB_DIR}/REPOS_NAME
    cd ${GB_DIR}/REPOS_NAME
    make init

## Build it

    make build

## Test it

    make test

## Deploy It

    make deploy

## Other Commands

    make help
